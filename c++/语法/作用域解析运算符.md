在C++中，`::` 是**作用域解析运算符**，主要用于以下几个方面：

## 1. **访问命名空间成员**
```cpp
namespace MyNamespace {
    int value = 10;
}

int main() {
    int x = MyNamespace::value;  // 访问命名空间中的成员
    return 0;
}
```

## 2. **访问类的静态成员**
```cpp
class MyClass {
public:
    static int staticValue;
    static void staticFunc();
};

int MyClass::staticValue = 5;  // 定义静态成员

int main() {
    int val = MyClass::staticValue;  // 访问静态成员
    MyClass::staticFunc();           // 调用静态函数
    return 0;
}
```

## 3. **定义类外成员函数**
```cpp
class MyClass {
public:
    void memberFunc();  // 声明
};

// 在类外定义成员函数
void MyClass::memberFunc() {
    // 函数实现
}
```

## 4. **访问全局作用域**
当局部变量与全局变量同名时：
```cpp
int value = 100;  // 全局变量

int main() {
    int value = 50;  // 局部变量
    
    cout << value;         // 输出 50（局部变量）
    cout << ::value;       // 输出 100（全局变量，使用 ::）
    return 0;
}
```

## 5. **继承中的基类成员访问**
```cpp
class Base {
public:
    void func();
};

class Derived : public Base {
public:
    void func() {
        Base::func();  // 调用基类的func
        // 其他代码
    }
};
```

## 在你的代码示例中：

```cpp
cpp_wrapper::StudentWrapper stu(...);
// ↑ 表示 StudentWrapper 在 cpp_wrapper 命名空间中

stu.print();  // 使用点号 . 访问对象的成员函数
c_student_print(stu.getCPtr());  // 调用全局C函数

c_do_callback(cpp_wrapper::cpp_callback, &stu);
// ↑ cpp_callback 是 cpp_wrapper 命名空间中的函数
```

## 总结
`::` 的主要作用是**明确指定某个标识符的作用域**，避免命名冲突，使代码更加清晰。它是C++中实现封装、命名空间和类作用域管理的重要工具。
