## 一、I/O 与引脚相关
### 1. IO Pin Count（总引脚数）
- **定义**：器件封装包含的所有物理引脚总数（含电源、地、配置、专用时钟、用户I/O）。
- **作用**：
  - 决定**外部接口规模**（外设、传感器、存储器、通信接口）。
  - 同型号不同封装引脚数不同（如 XC7A35T：FTG256 / CPG236）。

### 2. Available IOBs / User I/O（可用用户I/O）
- **定义**：可配置为通用输入/输出的**IOB (I/O Block)** 数量。
- **作用**：
  - 实际可用用户引脚数（减去电源、地、配置、专用时钟）。
  - 决定能接多少外设、LED、按键、总线等。

### 3. I/O Standard Support（I/O 标准支持）
- LVCMOS、LVTTL、SSTL、HSTL、LVDS、 differential 等。
- 决定电平兼容性、驱动能力、抗干扰、速度。

### 4. I/O Banks（I/O 分组）
- 器件I/O分多个Bank，每组独立供电、独立配置。
- 支持多电压域、混合电平接口。

### 5. Differential I/O Pairs（差分对数量）
- 高速、低噪声差分信号（LVDS、TMDS、DDR）。

## 二、逻辑资源（核心）
### 6. LUT Elements（查找表）
- **定义**：6输入LUT（7系列及以后），实现任意组合逻辑。
- **作用**：
  - FPGA**组合逻辑基础单元**。
  - 数量直接决定**逻辑复杂度上限**。
  - 可配置为**分布式RAM (Distributed RAM)** / 移位寄存器。

### 7. Flip-Flop (FF) / Register（触发器）
- **定义**：D触发器，存储1bit状态。
- **作用**：
  - 实现**时序逻辑、状态机、计数器、寄存器**。
  - 数量决定**并行度与时序容量**。

### 8. CLB / Logic Cells（逻辑单元）
- CLB = 多个Slice；Slice = LUT + FF + 进位链。
- 衡量整体**逻辑密度**。

---

## 三、片上存储资源
### 9. Block RAMs (BRAM)
- **定义**：专用36Kb双端口块RAM（带FIFO/校验）。
- **作用**：
  - **高速片上缓存、FIFO、缓冲区、查找表、系数存储**。
  - 比分布式RAM**容量大、延迟固定、功耗低**。

### 10. Ultra RAMs (URAM)
- UltraScale+ 新增：**4Kb / 288Kb** 大块DRAM式存储。
- **作用**：
  - **超大容量缓存**（AI特征图、视频帧、大数据缓冲）。
  - 密度远超BRAM、适合高带宽流式存储。

### 11. Distributed RAM（分布式RAM）
- LUT配置的小容量RAM（每LUT ~64bit）。
- 适合**小深度、高并行、随机访问**缓存/寄存器堆。

### 12. Total RAM Bits（总存储比特数）
- BRAM + URAM + Distributed RAM 总和。

---

## 四、数字信号处理（DSP）
### 13. DSP Slices / DSPs（DSP48E等）
- **定义**：硬核乘加单元（25×18乘法 + 48bit累加 + 预加）。
- **作用**：
  - **高速乘法、乘加、MAC、FIR、FFT、卷积、矩阵运算**。
  - 比LUT实现**快10~100倍、省大量资源**。
  - 支持级联形成**滤波器链、乘法器链**。

---

## 五、时钟与管理
### 14. CMT (Clock Management Tile)
- 含 **MMCM / PLL**：锁相环、混合模式时钟管理器。
- **作用**：
  - **时钟倍频/分频/移相/去抖**。
  - 生成高精度低抖动时钟（系统、外设、高速接口）。

### 15. Global Clock Buffers（全局时钟树）
- 低偏斜、高扇出全局时钟网络。

---

## 六、高速串行收发器（GT）
### 16. GT Transceivers（高速收发器）
- **定义**：GTP / GTX / GTH / GTY 等高速SerDes。
- **速率**：6.6Gb/s ~ 112Gb/s（系列不同）。
- **作用**：
  - **PCIe、10G/400G以太网、SRIO、USB3、DisplayPort、JESD204**。
  - 芯片间/板间/系统间**超高速串行通信**。

### 17. PCIe Hard Blocks（PCIe硬核）
- 集成PCIe Endpoint/Root Port（Gen3/4/5 x1/x2/x4/x8/x16）。
- 免逻辑实现高速PCIe接口。

---

## 七、特殊硬核与外设
### 18. XADC / Analog Mixed Signal（XADC）
- 双12bit 1MSPS ADC + 温度/电压传感器。
- 用于**模拟采集、监控、闭环控制、边界扫描**。

### 19. Processing System (PS)（SoC：Zynq / Versal）
- 集成ARM Cortex-A/R/M 硬核处理器。
- **FPGA+CPU异构**（嵌入式控制+硬件加速）。

### 20. AI Engine（AI引擎，Versal）
- 向量DSP阵列，面向**AI推理、5G、信号处理**。

### 21. HBM (High Bandwidth Memory)
- 集成高带宽内存（如 32GB HBM2，带宽~460GB/s）。
- 数据中心/AI/网络**超大带宽存储墙突破**。

---

## 八、物理与环境属性
### 22. Package Type（封装）
- **类型**：BGA、FBGA、CSP、QFP 等。
- **影响**：
  - PCB布局、散热、成本、引脚数、机械强度。

### 23. Speed Grade（速度等级）
- **标记**：-1 / -2 / -3 / -2L / -3E 等。
- **作用**：
  - 数字越大**最高频率越高、时序更松、价格更高**。
  - 带L=低功耗；E=扩展温度。

### 24. Temperature Grade（温度等级）
- C(商业0~85°C)、I(工业-40~100°C)、Q(军品级-40~125°C)。

### 25. Power Characteristics（功耗）
- 静态/动态功耗、典型/最大功耗。
- 决定**电源设计、散热、成本、可靠性**。

### 26. Configuration（配置）
- 支持：JTAG、SPI/BPI Flash、SelectMAP、PCIe配置。

---

## 九、安全与可靠性
### 27. Security Features
- AES-256、SHA、PUF（物理不可克隆函数）、安全启动。

### 28. SEU/EDAC（单粒子翻转/纠错）
- 航天/车载/工业**高可靠应用**。
