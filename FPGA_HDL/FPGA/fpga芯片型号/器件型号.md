在 **Vivado**（Xilinx）或 **Quartus**（Intel）中选择 **器件型号（Family/Device）** 是项目的基础，它直接决定了芯片的**资源、速度、功耗、封装、温度等级**。下面以主流的 **Xilinx (AMD)** 和 **Intel (Altera)** 为例，详细讲解型号规则与关键属性。

---

## 一、Xilinx (AMD) 型号命名规则（最常用）
### 1. 标准格式（以 7 系列 / UltraScale 为例）
`XC[系列][代][资源]-[速度][封装][引脚数][温度]`

**示例：`XC7K325T-2FFG900I`**

### 2. 分段解析
1. **前缀（厂商/等级）**
   - `XC`：标准商用（Commercial）
   - `XA`：汽车级（Automotive）
   - `XQ`：航天/军工级

2. **系列（Family）**
   - `S`：Spartan（低成本、入门）
   - `A`：Artix（低功耗、中端）
   - `K`：Kintex（高性能、平衡）
   - `V`：Virtex（超高性能、高端）

3. **代（Generation）**
   - `6`：Spartan-6
   - `7`：7 系列（28nm，最主流教学/工业）
   - `U`：UltraScale（20nm）
   - `US+`：UltraScale+（16/14nm）

4. **资源规模（Logic Capacity）**
   - `325T`：约 **325k 逻辑单元（LUT+FF）**
   - `T` = Transceiver（含高速收发器）
   - `LX`：纯逻辑（无收发器）

5. **速度等级（Speed Grade）**
   - `-1` / `-2` / `-3`
   - **数字越大 → 速度越快、时序越容易满足**
   - 带 `L` 为低功耗（如 `-2L`）

6. **封装 + 引脚数**
   - `FFG`：FBGA 无铅封装
   - `900`：900 引脚

7. **温度等级（Temperature）**
   - `C`：Commercial 0°C ~ 85°C
   - `I`：Industrial -40°C ~ 100°C
   - `Q`：Extended -40°C ~ 125°C

### 3. 常用教学型号
- **Artix-7（入门/毕设）**
  - `XC7A35T-1CPG236C`（Basys3）
  - `XC7A100T-2CSG324C`（Nexys A7）
- **Zynq-7000（SoC，带 ARM）**
  - `XC7Z020-1CLG484C`（最常用）

## 二、Intel (Altera) 型号命名规则
### 1. 标准格式
`EP[代][系列][资源][封装][引脚][温度][速度][N]`

**示例：`EP4CE10F17C8N`**

### 2. 分段解析
1. **前缀**
   - `EP`：标准
   - `5C`：Cyclone V

2. **系列**
   - `C`：Cyclone（低成本）
   - `A`：Arria（中端）
   - `S`：Stratix（高端）

3. **类型**
   - `E`：普通逻辑
   - `GX`：含高速收发器

4. **资源**
   - `10`：约 **10k LE（逻辑单元）**

5. **封装 + 引脚**
   - `F`：FBGA
   - `17` → 256 引脚（编码对应）

6. **温度 + 速度**
   - `C8`：Commercial + 速度等级 8
   - **Intel 规则：数字越小 → 速度越快**（与 Xilinx 相反）

7. **`N`**：无铅（RoHS）

---

## 三、型号中 **5 大核心属性**（选型必看）
### 1. **逻辑资源（最关键）**
- **Xilinx**：LUT、FF、DSP、BRAM、IO
  - `XC7A35T`：35k LUT、1.8M BRAM、90 DSP
- **Intel**：LE、M9K/M10K、DSP
- **选型**：
  - 入门：**35T ~ 100T**（足够做 CPU/计数器/接口）
  - 复杂：**K325T / V485T** 级别

### 2. **速度等级（时序收敛）**
- **Xilinx**：-1 < -2 < -3（越大越快）
- **Intel**：8 < 7 < 6（越小越快）
- 同型号**速度越高越贵**，但**时序更容易满足**

### 3. **温度等级（环境）**
- **C（商用）**：室内、消费电子
- **I（工业）**：-40°C ~ 100°C（工业控制、户外）
- **A/Q**：车规/军工

### 4. **封装与引脚数**
- **BGA（FBGA/FFG）**：主流、体积小、引脚多
- **TQFP**：直插、适合手工焊接（教学常用）
- 引脚数决定：**可接外设数量（LED/按键/以太网/HDMI）**

### 5. **工艺节点（决定功耗/性能）**
- 7 系列：**28nm**（最成熟、资料最多）
- UltraScale+：16/14nm（更低功耗、更高性能）
