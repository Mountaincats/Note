### Xilinx, Intel, AMD, Altera 的关系（这是行业并购史）

**过去：两大阵营分立（~2015年前）**
1.  **Xilinx** 和 **Altera** 是全球FPGA领域两大巨头，长期竞争。
2.  **Intel (英特尔)** 和 **AMD (超威半导体)** 是全球CPU领域两大巨头，长期竞争。

**并购潮：巨头吞并FPGA厂商（2015-2022年）**
1.  **Intel 收购 Altera (2015年完成)**
    *   英特尔为了进军数据中心、5G、AI加速等领域，斥巨资收购了Altera。
    *   此后，**Altera的FPGA产品线归属于Intel公司**，但品牌名在一段时间内仍保留“Altera”。现在官方已统一称为 **“Intel® FPGAs”**。所以你现在在官网看到的是 **Intel Quartus** 软件和 **Intel Agilex, Intel Stratix 10** 等FPGA系列。
    *   **简单记**：**Altera -> Intel**。

2.  **AMD 收购 Xilinx (2022年完成)**
    *   AMD为了增强在数据中心、高性能计算、自适应计算等领域的实力，收购了Xilinx。
    *   这是目前最新的重大并购。收购后，**Xilinx的FPGA产品线归属于AMD公司**。但与Intel不同，AMD在很大程度上保留了“Xilinx”这个极具影响力的品牌和技术名称。所以你现在看到的官方名称常常是 **“AMD Xilinx”** 或 **“AMD (formerly Xilinx)”**。软件Vivado、芯片系列（如Versal, Kintex）的名称都保持不变。
    *   **简单记**：**Xilinx -> AMD**。

**现在：新的竞争格局**
*   **AMD (Xilinx)** 对阵 **Intel (Altera)**。这两大CPU巨头，现在各自拥有了一家顶级FPGA公司，在高性能计算、数据中心、嵌入式等全方位领域展开竞争。

| 工具/芯片品牌 | 背后的母公司 | 说明 |
| :--- | :--- | :--- |
| **Vivado** | **AMD** | 开发工具，用于设计Xilinx/AMD的FPGA。 |
| **Xilinx 7系列, Zynq, UltraScale** | **AMD** | 芯片系列，现在属于AMD公司。 |
| **Quartus** | **Intel** | 开发工具，用于设计Intel(原Altera)的FPGA。 |
| **Intel Cyclone, Arria, Stratix** | **Intel** | 芯片系列，属于Intel公司。 |
