# C/C++ 混合编程 补充要点 + 完整入门实战项目

## 一、补充核心要点
### 1. 头文件兼容写法
C 和 C++ 共用头文件时，必须同时兼容：
- C 编译器（无 `extern "C"`）
- C++ 编译器（必须加 `extern "C"`）
- 防止头文件重复包含

标准写法：
```c
#ifndef MY_C_LIB_H
#define MY_C_LIB_H

// 只有 C++ 编译器才识别 __cplusplus 宏
#ifdef __cplusplus
extern "C" {
#endif

// 所有 C 函数、结构体声明放在这里

#ifdef __cplusplus
}
#endif

#endif
```

### 2. C++ 调用 C / C 调用 C++ 的区别
- **C++ 调用 C**：最常用，用 `extern "C"` 包含 C 头文件即可
- **C 调用 C++**：**只能调用被 extern "C" 修饰的 C++ 函数**，不能直接用 C++ 类/STL

### 3. 字符串互操作（高频坑点）
- C：`char*` / `const char*`（以`\0`结尾）
- C++：`std::string`
- 互转：`std::string.c_str()` → 传给 C 函数

### 4. 回调函数传参（实战必备）
C 语言回调无法直接传 C++ 对象，必须用 `void*` 透传对象指针。

### 5. 宏与关键字冲突
- C 没有 `namespace`/`class`/`new`/`delete`
- C 特有：**可变参数宏、指定初始化**
- 混合时不要用关键字做变量名

### 6. 编译与链接规则
- `.c` 文件：用 **C 编译器**（gcc）
- `.cpp` 文件：用 **C++ 编译器**（g++）
- 最终**统一用 g++ 链接**（兼容 C 标准库）

## 二、完整实战项目（覆盖所有细节）
### 项目功能
1. 编写一个 **C 语言工具库**（字符串、内存、计算）
2. 用 **C++ 封装 C 库**（RAII 资源管理、类封装）
3. 实现 **C++ 调用 C** + **C 调用 C++ 回调**
4. 包含：头文件兼容、内存管理、类型转换、错误处理

### 项目文件结构
```
c_cpp_mix/
├── c_lib/            # C 语言库
│   ├── c_utils.h     # C 头文件（兼容 C/C++）
│   └── c_utils.c     # C 实现
├── cpp_wrapper/      # C++ 封装层
│   ├── cpp_wrapper.h # C++ 类声明
│   └── cpp_wrapper.cpp # C++ 实现
├── main.cpp          # 主程序（入口）
└── Makefile          # 编译脚本
```

## 三、完整代码（逐行注释）
### 1. C 库头文件：c_utils.h（最标准的兼容写法）
```c
#ifndef C_UTILS_H
#define C_UTILS_H

// 核心：C++ 编译器自动加 extern "C"，C 编译器忽略
#ifdef __cplusplus
extern "C" {
#endif

// ==================== C 结构体 ====================
// 学生结构体（纯 C 风格，无 C++ 特性）
typedef struct {
    int id;
    char name[32];
    float score;
} CStudent;

// ==================== C 函数声明 ====================
// 创建学生（C 内存分配：malloc）
CStudent* c_student_create(int id, const char* name, float score);

// 释放学生（C 内存释放：free）
void c_student_free(CStudent* stu);

// 打印学生信息
void c_student_print(const CStudent* stu);

// 计算函数（纯 C 接口）
int c_add(int a, int b);

// C 语言回调：参数是 void* 通用指针
typedef void (*CCallback)(const char* msg, void* user_data);

// C 函数：接收回调并执行（用于 C 调用 C++ 函数）
void c_do_callback(CCallback cb, void* user_data);

#ifdef __cplusplus
}
#endif

#endif
```

### 2. C 库实现：c_utils.c
```c
#include "c_utils.h"
#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// 创建 C 学生对象
CStudent* c_student_create(int id, const char* name, float score) {
    // C 语言只能用 malloc
    CStudent* stu = (CStudent*)malloc(sizeof(CStudent));
    if (!stu) return NULL;

    // 赋值
    stu->id = id;
    strncpy(stu->name, name, sizeof(stu->name) - 1);
    stu->score = score;
    return stu;
}

// 释放 C 学生对象
void c_student_free(CStudent* stu) {
    if (stu) {
        free(stu); // 必须和 malloc 配对
    }
}

// 打印
void c_student_print(const CStudent* stu) {
    if (!stu) return;
    printf("[C Print] ID: %d, Name: %s, Score: %.1f\n",
           stu->id, stu->name, stu->score);
}

// 加法
int c_add(int a, int b) {
    return a + b;
}

// 执行回调
void c_do_callback(CCallback cb, void* user_data) {
    if (cb) {
        cb("Hello from C callback!", user_data);
    }
}
```

### 3. C++ 封装头文件：cpp_wrapper.h
```cpp
#ifndef CPP_WRAPPER_H
#define CPP_WRAPPER_H

// 包含 C 头文件（自动带 extern "C"）
#include "../c_lib/c_utils.h"
#include <string>

// C++ 命名空间（避免冲突）
namespace cpp_wrapper {

// ==================== RAII 封装类 ====================
// 自动管理 C 结构体内存：创建/自动释放，杜绝泄漏
class StudentWrapper {
private:
    CStudent* m_c_stu; // 持有 C 指针

public:
    // 构造：调用 C 函数创建对象
    StudentWrapper(int id, const std::string& name, float score);

    // 析构：自动调用 C 函数释放（核心！）
    ~StudentWrapper();

    // 禁止拷贝（防止重复释放）
    StudentWrapper(const StudentWrapper&) = delete;
    StudentWrapper& operator=(const StudentWrapper&) = delete;

    // 允许移动（高效转移所有权）
    StudentWrapper(StudentWrapper&& other) noexcept;

    // C++ 风格接口
    void print() const;
    float getScore() const;

    // 获取原始 C 指针（供外部 C 函数使用）
    CStudent* getCPtr() const { return m_c_stu; }
};

// ==================== C++ 回调函数 ====================
// 供 C 调用的 C++ 静态函数（必须 extern "C"）
extern "C" void cpp_callback(const char* msg, void* user_data);

} // namespace

#endif
```

### 4. C++ 封装实现：cpp_wrapper.cpp
```cpp
#include "cpp_wrapper.h"
#include <iostream>

namespace cpp_wrapper {

// 构造函数：C++ 创建 C 对象
StudentWrapper::StudentWrapper(int id, const std::string& name, float score) {
    // std::string → const char* 传给 C
    m_c_stu = c_student_create(id, name.c_str(), score);
}

// 析构函数：自动释放，无需手动调用 free
StudentWrapper::~StudentWrapper() {
    c_student_free(m_c_stu);
    std::cout << "[C++ RAII] C Student memory freed\n";
}

// 移动构造
StudentWrapper::StudentWrapper(StudentWrapper&& other) noexcept
    : m_c_stu(other.m_c_stu) {
    other.m_c_stu = nullptr; // 置空原对象，避免重复释放
}

// C++ 打印
void StudentWrapper::print() const {
    if (!m_c_stu) return;
    std::cout << "[C++] ID: " << m_c_stu->id
              << ", Name: " << m_c_stu->name
              << ", Score: " << m_c_stu->score << "\n";
}

float StudentWrapper::getScore() const {
    return m_c_stu ? m_c_stu->score : 0.0f;
}

// ==================== 回调实现 ====================
// extern "C" 让 C 可以调用这个 C++ 函数
void cpp_callback(const char* msg, void* user_data) {
    // 把 void* 转回 C++ 对象指针（类型安全）
    StudentWrapper* stu = static_cast<StudentWrapper*>(user_data);

    std::cout << "\n[C++ Callback] C msg: " << msg << "\n";
    std::cout << "[C++ Callback] Student score: " << stu->getScore() << "\n";
}

} // namespace
```

### 5. 主程序：main.cpp（项目入口）
```cpp
#include <iostream>
#include "cpp_wrapper/cpp_wrapper.h"

int main() {
    std::cout << "===== C/C++ 混合编程入门示例 =====\n\n";

    // ==================== 1. C++ 直接调用 C 函数 ====================
    std::cout << "1. C++ 调用 C 函数：c_add(10, 20) = "
              << c_add(10, 20) << "\n\n";

    // ==================== 2. C++ RAII 封装 C 对象 ====================
    std::cout << "2. C++ 封装 C 结构体（自动内存管理）\n";
    cpp_wrapper::StudentWrapper stu(1001, "ZhangSan", 95.5f);

    // C++ 风格调用
    stu.print();
    // 直接传给 C 函数
    c_student_print(stu.getCPtr());

    // ==================== 3. C 调用 C++ 回调函数 ====================
    std::cout << "\n3. C 调用 C++ 回调\n";
    c_do_callback(cpp_wrapper::cpp_callback, &stu);

    std::cout << "\n===== 程序结束，RAII 自动释放内存 =====\n";
    return 0;
}
```

### 6. 编译脚本：Makefile（一键编译）
```makefile
# 编译器
CC = gcc
CXX = g++

# 编译选项
CFLAGS = -c -Wall -g
CXXFLAGS = -c -Wall -g

# 目标文件
TARGET = c_cpp_mix

# 所有 .o 文件
OBJS = c_lib/c_utils.o \
       cpp_wrapper/cpp_wrapper.o \
       main.o

# 链接：必须用 g++（支持 C++ 标准库）
$(TARGET): $(OBJS)
	$(CXX) $(OBJS) -o $(TARGET)

# 编译 C 文件
c_lib/c_utils.o: c_lib/c_utils.c c_lib/c_utils.h
	$(CC) $(CFLAGS) c_lib/c_utils.c -o c_lib/c_utils.o

# 编译 C++ 文件
cpp_wrapper/cpp_wrapper.o: cpp_wrapper/cpp_wrapper.cpp cpp_wrapper/cpp_wrapper.h
	$(CXX) $(CXXFLAGS) cpp_wrapper/cpp_wrapper.cpp -o cpp_wrapper/cpp_wrapper.o

main.o: main.cpp cpp_wrapper/cpp_wrapper.h
	$(CXX) $(CXXFLAGS) main.cpp -o main.o

# 清理
clean:
	rm -f $(OBJS) $(TARGET)
```

## 五、项目覆盖的所有核心知识点
1. **`extern "C"` 完整用法**（解决名称修饰）
2. **C/C++ 共用头文件标准写法**
3. **内存管理**：`malloc/free` + `new/delete` 分离
4. **RAII 资源自动管理**（C++ 核心优势）
5. **字符串互转**：`std::string` ↔ `const char*`
6. **指针转换**：`void*` ↔ C++ 对象指针
7. **回调互操作**：C 调用 C++ 函数
8. **禁止拷贝/移动语义**（避免内存泄漏）
9. **编译链接规则**：C 用 gcc、C++ 用 g++、链接用 g++
