### C 语言运算符优先级与结合性

| 优先级 | 运算符                                                               | 名称/含义                     | 结合性             | 示例                                   |
| ------ | -------------------------------------------------------------------- | ----------------------------- | ------------------ | -------------------------------------- |
| **1**  | `()` `[]` `.` `->`                                                   | 函数调用/数组下标<br>成员访问 | 从左到右 (L-R)     | `func()` `arr[0]` `obj.mem` `ptr->mem` |
| **2**  | `++` `--` (后缀)                                                     | 后缀自增/自减                 | 从左到右 (L-R)     | `a++` `b--`                            |
|        | `+` `-` (一元)                                                       | 正号/负号                     | **从右到左 (R-L)** | `-5` `+value`                          |
|        | `++` `--` (前缀)                                                     | 前缀自增/自减                 | **从右到左 (R-L)** | `++i` `--j`                            |
|        | `!` `~`                                                              | 逻辑非/按位取反               | **从右到左 (R-L)** | `!flag` `~bits`                        |
|        | `*` (一元)                                                           | 解引用                        | **从右到左 (R-L)** | `*ptr`                                 |
|        | `&`                                                                  | 取地址                        | **从右到左 (R-L)** | `&var`                                 |
|        | `sizeof`                                                             | 数据类型大小                  | **从右到左 (R-L)** | `sizeof(int)`                          |
|        | `(type)`                                                             | 强制类型转换                  | **从右到左 (R-L)** | `(float)x`                             |
| **3**  | `*` `/` `%`                                                          | 乘/除/取模                    | 从左到右 (L-R)     | `a * b` `c / d` `e % f`                |
| **4**  | `+` `-` (二元)                                                       | 加/减                         | 从左到右 (L-R)     | `x + y` `z - 1`                        |
| **5**  | `<<` `>>`                                                            | 左移/右移                     | 从左到右 (L-R)     | `num << 2` `bits >> 1`                 |
| **6**  | `<` `<=` `>` `>=`                                                    | 关系运算符                    | 从左到右 (L-R)     | `a < b` `c >= d`                       |
| **7**  | `==` `!=`                                                            | 等于/不等于                   | 从左到右 (L-R)     | `x == y` `z != 0`                      |
| **8**  | `&`                                                                  | 按位与                        | 从左到右 (L-R)     | `flags & MASK`                         |
| **9**  | `^`                                                                  | 按位异或                      | 从左到右 (L-R)     | `a ^ b`                                |
| **10** | `\|`                                                                 | 按位或                        | 从左到右 (L-R)     | `x \| y`                               |
| **11** | `&&`                                                                 | 逻辑与                        | 从左到右 (L-R)     | `cond1 && cond2`                       |
| **12** | `\|\|`                                                               | 逻辑或                        | 从左到右 (L-R)     | `A \|\| B`                             |
| **13** | `?:`                                                                 | 条件运算符                    | **从右到左 (R-L)** | `a > b ? x : y`                        |
| **14** | `=` `+=` `-=` `*=`<br> `/=` `%=` `&=`<br> `^=` `\|=` `<<=`<br> `>>=` | 赋值与复合赋值                | **从右到左 (R-L)** | `x = y` `a += 5`                       |
| **15** | `,`                                                                  | 逗号运算符                    | 从左到右 (L-R)     | `a = 1, b = 2`                         |

---
C语言中有几种不同的命名空间：

1.​​标签名（label names）​​：用于goto语句的标签。

2.​​结构体、联合体和枚举的标签（struct, union, and enum tags）​​：例如，struct foo中的foo就是标签。

3.​​结构体和联合体的成员名（members of struct or union）​​：每个结构体或联合体都有独立的命名空间，因此不同结构体的成员名可以相同。

4.​​普通标识符（ordinary identifiers）​​：包括变量名、函数名、类型定义名（typedef）和枚举常量（enumeration constants）。
