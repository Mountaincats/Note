## 一、应用窗口切换

在Ubuntu系统中有几种非常高效的方法可以在不同应用间快速切换。无需用鼠标点击Dock栏，通过键盘快捷键即可实现，这能极大提升多任务处理效率

以下是几个最核心、最实用的快捷键：

### 1. **核心应用切换：`Alt + Tab`**
   - **功能**：在所有**正在运行的应用窗口**之间循环切换
   - **操作**：按住 `Alt` 键不放，然后重复按 `Tab` 键，可以在所有打开的窗口（包括不同应用和同一应用的不同窗口）间向前切换。如果同时按住 `Shift` 键（即 `Alt + Shift + Tab`），则向后切换
   - **提示**：这是最通用、最符合直觉的方法，适合在任意两个应用间跳转，比如从VS Code直接切换到PDF阅读器

### 2. **超级键预览：`Super` 键**
   - **功能**：按下键盘上的 `Super` 键（通常带有Windows徽标）可以**显示所有窗口的概览**。此时，您可以直接用方向键选择，或**用鼠标点击**要切换的窗口
   - **操作**：按下 `Super` 后，所有窗口会平铺显示，您能直观地看到所有打开的应用，包括PDF阅读器。这是一种“可视化”的切换方式

### 3. **同一应用内窗口切换：`` Alt + ` ``（反引号键）**
   - **功能**：如果您打开了**同一个应用的多个窗口**（例如，多个VS Code窗口或多个PDF文件），此快捷键可以在**该应用的窗口间**快速切换，而不会切换到其他应用
   - **操作**：先按住 `Alt` 键，然后按 `` ` `` 键（通常在 `Tab` 键上方，数字1左边）

### 核心技巧总结：
- **在任意两个应用间切换**：使用 **`Alt + Tab`**，这是最推荐的习惯
- **想看清所有窗口再选**：按一下 **`Super`** 键
- **在同一个软件的多个窗口间切换**：使用 **`` Alt + ` ``**

## 二、应用窗口关闭(挺多快捷键不通用，了解下即可，但最小化窗口通用)

通过键盘快捷键关闭应用窗口非常快捷。根据是想“优雅地退出”还是“强制关闭”，有以下几种常用方法：

### 1. 常规关闭（推荐）

这些方法会正常关闭窗口，给应用程序保存数据的机会。

1.  **关闭当前活动窗口： `Alt + F4`**
    *   **功能**：这是 **最通用、最经典** 的快捷键，可以关闭当前获得焦点的任何窗口（应用、文件管理器、设置窗口等）
    *   **操作**：确保要关闭的窗口在最前面（比如您的VS Code或PDF阅读器），然后按下 `Alt + F4`

2.  **使用应用程序自身菜单： `Alt + F`**
    *   **功能**：大多数应用程序的菜单栏中都有“文件(File)”菜单，其中包含“退出(Quit)”或“关闭窗口(Close Window)”选项
    *   **操作**：按 `Alt + F` 打开“文件”菜单，然后按方向键选择“退出(Quit)”，最后按 `Enter` 确认
3.  **在许多应用中，可以直接按 `Ctrl + Q`（这是许多Linux/跨平台应用的通用退出快捷键**
    *   如VS Code、Firefox、LibreOffice等

### 2. 针对Dock栏上已固定应用

如果您想快速关闭一个固定在Dock栏上的应用（例如VS Code），可以利用它的位置编号

*   **关闭特定应用窗口： `Super + Alt + 数字`**
    *   **功能**：按住 `Super + Alt`，然后按Dock栏上对应应用的**数字键**，可以显示该应用的窗口菜单，其中包含“**关闭所有窗口(Quit)**”的选项
    *   **操作**：例如，如果VS Code是您Dock栏从上往下数的第6个图标，那么按 `Super + Alt + 6`，然后按方向键选择“退出”或直接按 `Q`（如果菜单支持快捷键）

### 3. 最小化与隐藏

有时您可能不想关闭，只是想暂时隐藏窗口

*   **最小化当前窗口： `Super + H`**
    *   **功能**：将当前窗口最小化到Dock栏。这比关闭更轻量，可以随时再点开
*   **隐藏所有窗口显示桌面： `Super + D`**
    *   **功能**：一键最小化所有窗口，显示干净的桌面

### 4. 强制关闭（应用无响应时使用）

当某个应用卡死、失去响应时，上面的方法可能无效，这时需要使用系统监视器或终端命令

1.  **调用系统监视器： `Ctrl + Alt + Del`** （和Windows一样）
    *   **功能**：会弹出一个菜单，选择“**活动进程(Processes)**”或直接打开**系统监视器(System Monitor)**，找到无响应的应用，右键选择“结束进程(Kill Process)”
2.  **使用终端命令 `xkill`**
    *   **功能**：一个非常直接的工具，让你用鼠标点选哪个窗口需要被“杀死”
    *   **操作**：
        1.  按 `Ctrl + Alt + T` 打开终端
        2.  输入命令 `xkill` 然后按回车
        3.  鼠标光标会变成一个“骷髅”或“十字”图标
        4.  **用这个图标点击**那个无响应的窗口，该窗口会立即被强制关闭
    *   **提示**：这个*方法只关闭窗口，不一定会让整个应用完全退出

### 总结与建议

| 场景 | 推荐快捷键 | 说明 |
| :--- | :--- | :--- |
| **正常关闭当前窗口** | **`Alt + F4`** | **最常用，首选** |
| **退出当前应用（如VS Code）** | **`Ctrl + Q`** | 很多图形应用的通用退出键 |
| **最小化当前窗口** | `Super + H` | 暂时隐藏，而非关闭 |
| **关闭固定应用** | `Super + Alt + 数字` | 结合Dock栏位置使用 |
| **应用无响应** | `Ctrl + Alt + Del` -> 活动进程 | 或使用终端命令 `xkill` |

对于日常使用，掌握 **`Alt + F4`** 和 **`Ctrl + Q`** 就足够了。前者是“系统级”的关闭窗口命令，后者是许多“应用程序内部”的退出命令。两者结合，可以高效地管理所有窗口
