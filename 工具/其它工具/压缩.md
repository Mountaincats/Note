# 文件压缩与解压命令参考

## 快速查阅表格

| 格式 | 压缩命令 | 解压命令 |
|------|----------|----------|
| .zip | zip -r 文件名.zip 目录 | unzip 文件名.zip |
| .tar | tar -cvf 文件名.tar 目录 | tar -xvf 文件名.tar |
| .tar.gz | tar -czvf 文件名.tar.gz 目录 | tar -xzvf 文件名.tar.gz |
| .tar.bz2 | tar -cjvf 文件名.tar.bz2 目录 | tar -xjvf 文件名.tar.bz2 |
| .tar.xz | tar -cJvf 文件名.tar.xz 目录 | tar -xJvf 文件名.tar.xz |
| .7z | 7z a 文件名.7z 目录 | 7z x 文件名.7z |
| .rar | rar a 文件名.rar 目录 | unrar x 文件名.rar |
| .gz | gzip 文件名 | gzip -d 文件名.gz |

---

## ZIP

### 压缩
* 命令：`zip -r 压缩包名.zip 要压缩的文件或目录`

### 常用参数：
* -r：递归压缩目录
* -q：安静模式（不显示压缩过程）
* -e：加密压缩（会提示输入密码）
* -1 ~ -9：压缩级别（1最快压缩率低，9最慢压缩率高）
* -x：排除某些文件

* 示例：
```
zip -r backup.zip /home/user/docs
zip -r -9 project.zip src/ -x "*.log"
```

### 解压
* 命令：`unzip 压缩包.zip`

### 常用参数：
* -d：解压到指定目录
* -l：查看压缩包内容（不解压）
* -o：覆盖已存在文件不提示
* -q：安静模式
 
* 示例：
```
unzip data.zip -d /target/path
unzip -l archive.zip
```

---

## TAR

### 打包（不压缩）
* 命令：`tar -cvf 打包文件.tar 文件或目录`

### 解包
* 命令：`tar -xvf 打包文件.tar`

### 常用参数：
* -c：创建打包文件
* -x：解包
* -v：显示详细过程
* -f：指定打包文件名
* -t：查看包内文件（不解压）
* -C：解压到指定目录

---

## TAR + GZIP (.tar.gz 或 .tgz)

### 压缩
* 命令：`tar -czvf 打包文件.tar.gz 文件或目录`

### 解压
* 命令：`tar -xzvf 打包文件.tar.gz`

### 常用参数：
* -z：调用 gzip 压缩/解压

---

## TAR + BZIP2 (.tar.bz2 或 .tbz2)

### 压缩
* 命令：`tar -cjvf 打包文件.tar.bz2 文件或目录`

### 解压
* 命令：`tar -xjvf 打包文件.tar.bz2`

### 常用参数：
* -j：调用 bzip2 压缩/解压

---

## TAR + XZ (.tar.xz)

### 压缩
* 命令：`tar -cJvf 打包文件.tar.xz 文件或目录`

### 解压
* 命令：`tar -xJvf 打包文件.tar.xz`

### 常用参数：
* -J：调用 xz 压缩/解压

---

## GZIP

### 压缩
* 命令：`gzip 文件名`
* 说明：生成 文件名.gz（会删除原文件）
* 保留原文件压缩：
`gzip -c 原文件 > 压缩文件.gz`

### 解压
* 命令：`gzip -d 文件名.gz` 或 `gunzip 文件名.gz`

### 常用参数：
* -J：调用 xz 压缩/解压

---

## 7Z

### 压缩
* 命令：`7z a 压缩包.7z 文件或目录`

### 解压
* 命令：`7z x 压缩包.7z`

### 常用参数：
* a：添加文件到压缩包
* x：解压（保留目录结构）
* -t：指定压缩格式
* -p：设置密码

---

## RAR

### 压缩
* 命令：`rar a 压缩包.rar 文件或目录`

### 解压
* 命令：`unrar x 压缩包.rar`

### 常用参数：
* a：添加文件
* x：解压（保留路径）
* e：解压到当前目录（不保留路径）
